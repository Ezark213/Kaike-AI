## はじめに

個人事業主やフリーランスが最も悩むポイントの一つが「これは経費になるのか？」という税務判定です。交際費と私的な食事の境界線、自宅兼事務所の家事按分、書籍やセミナーの経費算入可否など、判断に迷うケースは日常的に発生します。

本記事では、ChatGPTを活用して経費の税務判定を効率化する3つの実践的なプロンプトを紹介します。AIの回答をそのまま鵜呑みにするのではなく、判断の「たたき台」として活用し、最終確認は税理士に委ねるワークフローを推奨します。

## プロンプト1: 経費該当性の総合判定

最も汎用的なプロンプトです。取引内容を入力すると、経費として認められるかどうかを複数の観点から判定します。

```
あなたは日本の税務に詳しい経理アドバイザーです。
以下の取引について、個人事業主（所得税法）の観点から
経費（必要経費）として認められるか判定してください。

【判定対象の取引】
- 事業内容: {あなたの事業内容}
- 取引の内容: {何を購入/支払ったか}
- 金額: {金額}
- 日時: {日時}
- 取引の目的・経緯: {なぜこの支出が必要だったか}

【回答形式】
1. 判定結果: ○経費になる / △一部経費になる / ×経費にならない
2. 該当する勘定科目（候補を最大3つ）
3. 判定根拠（所得税法の条文や通達を引用）
4. 按分が必要な場合、推奨按分率と根拠
5. 税務調査で指摘されるリスク（高/中/低）
6. リスクを下げるための証拠書類・記録のアドバイス
```

例えば「Webデザイナーがクライアントとの打ち合わせ後に食事（2名分、¥8,500）」と入力すると、判定結果（○経費になる）、勘定科目（接待交際費）、判定根拠（所得税法第37条）、税務調査リスク（中）、証拠書類のアドバイス（領収書に参加者名記録、議事メモ作成等）が得られます。

## プロンプト2: 家事按分の適正率算出

自宅兼事務所の家賃、電気代、通信費など、事業とプライベートで共用する支出の按分率を算出するプロンプトです。

```
あなたは日本の所得税法に詳しい税務アドバイザーです。
以下の情報をもとに、家事按分の適正な事業使用割合を算出してください。

【基本情報】
- 事業内容: {事業内容}
- 業務形態: {在宅勤務/外出あり/完全在宅}
- 1日の平均業務時間: {時間}
- 週の稼働日数: {日数}

【按分対象の支出】
- 支出項目: {家賃/電気代/通信費/車両費 等}
- 月額: {金額}
- 使用状況の詳細: {具体的な使用状況}

【住居情報】（家賃・光熱費の場合）
- 総面積: {㎡}
- 事業専用スペースの面積: {㎡}
- 共用スペースの面積: {㎡}

【回答形式】
1. 推奨按分率（%）
2. 算出方法（面積按分/時間按分/使用量按分）
3. 算出根拠の詳細計算
4. 税務調査で認められやすい按分率の範囲
5. 按分率を裏付けるために準備すべき資料
```

例えば「完全在宅プログラマー、インターネット通信費¥5,500/月、業務8h/日×5日」と入力すると、推奨按分率（60〜70%）、算出方法（時間按分＋通信量比率）、税務調査で認められやすい範囲（50〜70%）、準備すべき資料（業務日報、通信量ログ等）が得られます。

## プロンプト3: 一括経費判定（月次レシートまとめて判定）

月末にまとめてレシートを整理する際に便利な、複数の支出を一括で判定するプロンプトです。

```
あなたは日本の税務に詳しい経理アドバイザーです。
以下の支出一覧について、各項目の経費該当性を一括判定してください。

【事業者情報】
- 事業内容: {事業内容}
- 青色申告/白色申告: {どちらか}
- 消費税: {課税事業者/免税事業者}

【支出一覧】
| No | 日付 | 内容 | 金額 | メモ |
|----|------|------|------|------|
| 1 | {日付} | {内容} | {金額} | {補足} |
| 2 | {日付} | {内容} | {金額} | {補足} |
...

【回答形式】
各項目について以下の表形式で回答してください：
| No | 判定 | 勘定科目 | 按分率 | リスク | 備考 |

判定は ○=全額経費 / △=一部経費 / ×=経費不可 / ?=情報不足
最後に、特に注意が必要な項目について詳細コメントをください。
```

事業者情報と支出一覧を表形式で入力すると、各項目に対して判定（○/△/×）・勘定科目・按分率・リスク・備考を表形式で一括回答してくれます。例えば技術書は○新聞図書費、1名での食事は×、コワーキングスペースは○地代家賃、といった具合です。

## プロンプト活用のコツ

### 1. 事業内容を具体的に書く

「IT系フリーランス」ではなく「React/Next.jsを使ったWeb開発をメインとするフリーランスエンジニア。クライアントは中小企業が中心」のように具体的に記述すると、判定精度が向上します。

### 2. 取引の経緯を詳しく書く

「なぜこの支出が事業に必要だったか」の文脈情報が多いほど、AIは正確な判定ができます。特に、交際費やグレーゾーンの支出では、目的・経緯・参加者の情報が重要です。

### 3. 判定結果をスプレッドシートで管理する

ChatGPTの判定結果をそのまま確定申告に使うのではなく、スプレッドシートに記録して税理士のレビューに回すワークフローがお勧めです。

```
【月次レビューフロー】
1. 週次: レシートを撮影・収集
2. 月末: ChatGPTで一括判定（プロンプト3）
3. 判定結果をスプレッドシートに転記
4. △や?の項目をリストアップ
5. 税理士に△/?項目のみ確認依頼
6. 確定した仕訳をfreeeまたはMFに入力
```

## AIの判定を信頼しすぎないために

ChatGPTの税務判定は非常に有用ですが、以下の限界を理解しておく必要があります。

1. **法改正への追従**: AIの学習データに最新の税法改正が反映されていない場合がある
2. **個別事情の考慮**: 業種特有の事情や過去の税務調査での指摘事項は考慮されない
3. **グレーゾーンの判断**: 税務上のグレーゾーンでは保守的な判断をする傾向がある
4. **判例の網羅性**: 全ての関連判例を網羅しているとは限らない

AIは「判断の下書き」を作るツールとして活用し、最終判断は必ず専門家に委ねましょう。
