## はじめに

損益計算書（P/L）の分析は、月次経営報告や決算説明の中で最も重要な業務の一つです。ChatGPTに財務データを入力するだけで、プロ品質の分析コメントを自動生成できます。

---

## Tips 1：月次P/L分析レポートの自動生成

```
以下の損益計算書データをもとに、経営者向けの月次分析レポートを作成してください。

【月次データ】
（単位：万円）
               当月実績   前月実績   前年同月   予算
売上高:          [金額]    [金額]    [金額]    [金額]
売上原価:         [金額]    [金額]    [金額]    [金額]
売上総利益:        [金額]    [金額]    [金額]    [金額]
販管費:           [金額]    [金額]    [金額]    [金額]
営業利益:          [金額]    [金額]    [金額]    [金額]

【業種】: [業種を入力]

【レポートに含めてほしい内容】
1. 今月のハイライト（良かった点・気になる点を各1〜2文で）
2. 前月比で最も変動した項目とその推定要因
3. 予算との乖離で特に注意すべき項目
4. 来月への改善アクション（具体的に3つ）

経営者が5分で読めるレポートにしてください。
```

## Tips 2：売上原価率・粗利率の傾向分析

```
以下の月次推移データから、売上原価率と粗利率のトレンドを分析してください。

【月次推移（直近6ヶ月）】
        1月    2月    3月    4月    5月    6月
売上高:  [値]   [値]   [値]   [値]   [値]   [値]
売上原価: [値]   [値]   [値]   [値]   [値]   [値]
粗利:    [値]   [値]   [値]   [値]   [値]   [値]

【分析してほしい内容】
1. 粗利率の推移グラフの言語化（上昇・下降・安定の傾向）
2. 粗利率が最も良かった月・悪かった月の要因推定
3. このトレンドが続いた場合の12月末粗利率の予測
4. 粗利率改善のための具体的な提案（価格戦略・原価削減）
5. 同業種の一般的な粗利率水準との比較コメント
```

## Tips 3：販管費の内訳分析と削減提案

```
以下の販管費内訳データを分析し、コスト削減の機会を特定してください。

【販管費内訳（当月）】
人件費: [金額]
賃借料: [金額]
通信費: [金額]
広告宣伝費: [金額]
旅費交通費: [金額]
接待交際費: [金額]
消耗品費: [金額]
外注費: [金額]
その他: [金額]
合計: [金額]
売上高: [金額]

【分析してほしい内容】
1. 売上高に対する各費目の比率と、業種平均との比較
2. 特に割合が高い費目の詳細確認事項
3. 固定費と変動費への分類
4. 売上が20%減少した場合の損益分岐点への影響
5. 3〜6ヶ月以内に削減可能な費目と、削減額の目安
```

## Tips 4：赤字転落の要因分析と回復プラン

月次で赤字が出た場合の緊急分析プロンプトです。

```
今月の損益が赤字になりました。原因を分析し、回復プランを作成してください。

【損益データ】
売上高: [金額]（予算: [金額]）
売上原価: [金額]（予算: [金額]）
売上総利益: [金額]
販管費: [金額]（予算: [金額]）
営業損失: -[金額]

【赤字に至った背景（分かる範囲で記入）】
・売上の変化: [例：主要顧客Aが発注を一時停止]
・コストの変化: [例：原材料費が高騰]
・特別な要因: [例：設備修繕費が発生]

【分析してほしいこと】
1. 赤字の主要因（売上減少 or コスト増加の寄与度）
2. 一時的な要因 vs 構造的な要因の切り分け
3. 来月以降に同様の赤字を防ぐための具体的アクション（3つ）
4. 損益分岐点に戻るために必要な売上増加額または固定費削減額
5. 経営者への報告用の1段落サマリー
```

## Tips 5：部門別損益の可視化・比較

```
以下の部門別損益データを分析し、経営会議用の報告資料のテキスト版を作成してください。

【部門別損益（今月）】
         売上高  売上原価  粗利  販管費  営業利益  営業利益率
A事業部: [値]   [値]    [値]  [値]   [値]    [値]%
B事業部: [値]   [値]    [値]  [値]   [値]    [値]%
C事業部: [値]   [値]    [値]  [値]   [値]    [値]%
全社合計: [値]  [値]    [値]  [値]   [値]    [値]%

【分析してほしいこと】
1. 最も貢献度が高い部門・最も改善が必要な部門
2. 各部門の粗利率と営業利益率の評価コメント
3. 全社の利益を10%改善するために、各部門が取るべきアクション
4. 部門間でリソース（人・コスト）の配分を見直す場合の提案
5. 経営会議での報告用サマリー（3分で話せる内容）
```

## freee・MFでの損益データのエクスポート方法

### freeeの場合
1. 会計 → 試算表 → 月次損益計算書
2. 右上「ダウンロード」→ CSV形式で保存
3. ExcelまたはテキストでChatGPTに貼り付け

### MFの場合
1. 会計 → レポート → 損益計算書
2. 「エクスポート」→ CSV/Excel形式
3. ChatGPTに貼り付けて分析
