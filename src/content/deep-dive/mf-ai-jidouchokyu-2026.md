## 設定を最適化して精度を向上させる方法

MFクラウド会計のAI自動仕訳は「仕訳ルール」の設定が精度を大きく左右します。

### 推奨設定手順

**Step 1: 取引先マスタの整備**
```
設定 → 取引先マスタ → 取引先ごとの「デフォルト勘定科目」を設定
例: Amazon → 消耗品費 / ガス代 → 水道光熱費
```

**Step 2: 自動仕訳ルールのカスタマイズ**
```
設定 → 自動仕訳ルール → 「キーワード」「取引先」「金額範囲」で仕訳ルール追加
例: 備考に「会議」→ 会議費 / 金額5,000円以上で取引先が飲食店 → 交際費
```

**Step 3: フィードバックによる学習促進**
```
自動仕訳結果を確認 → 誤仕訳には「×」ボタンで訂正
→ AIが学習して同パターンの次回精度が向上
```

**CPA試験合格者監修コメント**:
初期設定に2〜3時間をかけることで、以降の月次経理を大幅に効率化できます。特に「仕訳ルール」の充実が投資対効果が最も高い設定です。会計士・税理士が顧問についている場合は、担当者と一緒に初期設定を行うことを強く推奨します。
