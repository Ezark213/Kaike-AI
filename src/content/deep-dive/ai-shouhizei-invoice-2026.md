## インボイス制度の会計処理

### AI×インボイスチェック

```
以下の請求書がインボイス（適格請求書）の要件を満たしているかチェックしてください。

【請求書の記載内容】
発行者: 株式会社○○商事
登録番号: T1234567890123
発行日: 2026年3月15日
品目: 事務用品一式
金額: ¥50,000
消費税: ¥5,000
合計: ¥55,000
支払期限: 2026年4月30日

【インボイスの記載要件（6項目）】
1. 適格請求書発行事業者の氏名/名称 + 登録番号
2. 取引年月日
3. 取引内容（軽減税率対象の場合はその旨）
4. 税率ごとに区分した対価の額（税抜/税込）+ 適用税率
5. 税率ごとに区分した消費税額
6. 書類の交付を受ける事業者の氏名/名称

各項目について PASS/FAIL で判定し、不備がある場合は修正方法を示してください。
```

### 課税区分の判定

```
以下の取引の消費税区分を判定してください。

【取引一覧】
1. 国内の事務所で使用するコピー用紙の購入 ¥5,000
2. 海外のクライアントへのコンサルティングサービス提供 ¥500,000
3. 社員の通勤定期券（電車）の購入 ¥30,000
4. 社宅の家賃（従業員負担分を除く）¥100,000
5. 海外出張時のホテル代 ¥50,000
6. 社会保険料の会社負担分 ¥200,000
7. 銀行の振込手数料 ¥880
8. 顧客への贈答品（お中元）¥10,000
9. 従業員への慶弔見舞金 ¥30,000
10. 外注先（個人・免税事業者）への業務委託料 ¥100,000

各取引について:
1. 課税区分（課税/非課税/不課税/免税）
2. 判定根拠（消費税法の該当条文）
3. インボイスの要否
4. 仕入税額控除の可否
5. 経過措置（免税事業者からの仕入れ）の適用有無
```

## 簡易課税 vs 本則課税の有利判定

```
以下のデータで、簡易課税と本則課税のどちらが有利か判定してください。

【事業データ】
- 業種: ソフトウェア開発（サービス業）
- 課税売上高: ¥30,000,000
- 課税仕入等の額:
  - 外注費: ¥8,000,000（課税仕入）
  - 地代家賃: ¥2,400,000（課税仕入）
  - 通信費: ¥600,000（課税仕入）
  - 消耗品費: ¥500,000（課税仕入）
  - 交通費: ¥300,000（課税仕入）
  - 接待交際費: ¥200,000（課税仕入）
  - 水道光熱費: ¥180,000（課税仕入）
  - 人件費: ¥15,000,000（不課税）
  - 保険料: ¥500,000（非課税）

【計算してほしい項目】
1. 本則課税の場合の消費税額
2. 簡易課税の場合の消費税額（第5種事業・みなし仕入率50%）
3. 有利判定の結果
4. 差額
5. 来期も同じ判断でよいかの目安
6. 届出期限の確認
```

## 2割特例の適用判断

```
以下の事業者が2割特例を適用できるか判定してください。

【事業者情報】
ケースA: フリーランスWebデザイナー
- 2023年10月にインボイス登録（免税→課税転換）
- 基準期間（2024年）の課税売上高: ¥8,000,000
- 当期（2026年）の課税売上高: ¥12,000,000

ケースB: 個人の翻訳者
- 2023年10月にインボイス登録
- 基準期間の課税売上高: ¥900,000
- 当期の課税売上高: ¥1,500,000

ケースC: 一人法人（コンサルティング）
- 2024年1月設立と同時にインボイス登録
- 資本金: ¥1,000,000
- 当期の課税売上高: ¥20,000,000

【各ケースについて回答】
1. 2割特例の適用可否
2. 適用要件の充足判定
3. 2割特例の消費税額
4. 本則課税/簡易課税との比較
5. 2割特例がいつまで使えるか
```
