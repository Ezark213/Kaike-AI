## 背景：固定資産管理の課題

固定資産管理は「地味だが間違えると税務リスクが大きい」業務です。耐用年数の選定ミス、資本的支出の判断ミス、除却漏れなどは税務調査で指摘されやすいポイントです。

**よくある問題**:

| 問題 | リスク | 頻度 |
|------|--------|------|
| 耐用年数の選定ミス | 過大/過小償却 → 税務修正 | 取得時 |
| 資本的支出vs修繕費の誤判定 | 一括損金 → 否認・追徴課税 | 修繕発生時 |
| 除却処理の漏れ | 存在しない資産に減価償却 → 過大費用計上 | 通年 |
| 少額資産の判定ミス | 即時償却/3年均等/通常償却の選択ミス | 取得時 |
| 固定資産台帳と現物の不一致 | 実在性の問題 → 監査指摘 | 期末棚卸し時 |

## AI固定資産管理の実践手法

### 1. 減価償却の完全自動計算

クラウド会計ソフトでは、固定資産を登録するだけで月次の減価償却費が自動計算されます。

**freee会計の固定資産管理**:
- 資産登録時に取得価額・取得日・耐用年数・償却方法を入力
- 月次の減価償却仕訳を自動生成（手動仕訳不要）
- 期末の減価償却累計額・帳簿価額を自動更新
- 固定資産台帳・減価償却明細表をワンクリック出力

**マネーフォワードの固定資産管理**:
- 同様の自動計算機能に加え、CSV一括登録に対応
- 償却方法の変更（定額法↔定率法）にも自動対応
- リース資産の管理にも対応（IFRS16対応含む）

### 2. 資本的支出vs修繕費の判定

```
以下の支出が資本的支出と修繕費のどちらに該当するか判定してください。

【支出内容】
1. 工場の屋根の全面張替え: ¥8,000,000
   - 旧屋根は築30年で雨漏り発生
   - 新しい屋根は断熱性能が従来より向上

2. オフィスのエアコン交換: ¥1,500,000
   - 故障した10年使用のエアコンを同等品に交換
   - 性能は従来と同等

3. 社用車のエンジン載せ替え: ¥800,000
   - 走行距離15万kmで故障
   - 新型エンジンで燃費が20%向上

4. 事務所のLED照明への交換: ¥500,000
   - 従来の蛍光灯をLEDに全面交換
   - 消費電力50%削減・寿命3倍

【各項目について回答してほしい内容】
1. 資本的支出/修繕費の判定結果
2. 判定根拠（法人税法基本通達の該当条文）
3. 20万円基準・60万円基準・形式基準の適用可否
4. 迷った場合の実務上の安全策
5. 税務調査で指摘されやすいポイント
```

```
3. サーバーラック用のコンピュータサーバー

7. 太陽光発電設備（自家消費用）

【回答に含めてほしい情報】
1. 耐用年数（法定耐用年数）
2. 該当する資産区分（建物/構築物/機械装置/器具備品/車両運搬具）
3. 償却方法（定額法/定率法の選択肢）
4. 中古資産の場合の見積耐用年数の計算方法
5. 少額減価償却資産（30万円未満）の特例適用可否
```
