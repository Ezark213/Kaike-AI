## 活用法①：仕訳の勘定科目チェック

freeeのAI自動仕訳は高精度ですが、判断が難しい取引は人間の確認が必要です。ChatGPTを「第二の目」として活用しましょう。

### プロンプト例：勘定科目の妥当性チェック

```
あなたは日本の税務に詳しい公認会計士です。
以下のfreee仕訳データの勘定科目が正しいかチェックしてください。
誤りがあれば正しい科目と根拠（税法・通達の条文番号）を示してください。

【仕訳データ】
日付: 2026-02-15
摘要: Amazon Business - ノートPC HP ProBook
金額: 198,000円
勘定科目: 消耗品費
税区分: 課税仕入10%

【事業情報】
- 個人事業主（青色申告）
- 事業: Webデザイン業
```

### ChatGPTの回答例

> 金額が10万円以上のため「消耗品費」ではなく以下の処理が必要です:
> - **10万円以上20万円未満**: 「一括償却資産」として3年均等償却（所得税法施行令第139条）
> - **10万円以上30万円未満**: 青色申告の場合「少額減価償却資産の特例」で全額経費（租税特別措置法第28条の2）
> - **推奨**: 少額減価償却資産の特例を適用し「減価償却費」198,000円を一括計上

**CPA試験合格者監修コメント**:
10万円超の資産の経費処理はfreeeのAI自動仕訳でも誤りやすいポイントです。ChatGPTに「青色申告」「個人事業主」等の前提条件を明示することで、より正確な回答が得られます。ただし、少額減価償却資産の特例は年間合計300万円の上限がある点は、ChatGPTが見落とすことがあるので注意してください。

## 活用法②：経費分類の判断サポート

個人事業主が最も悩むのが「これは経費にできるか？」という判断です。ChatGPTに具体的な状況を伝えることで、判断の参考意見を得られます。

### プロンプト例：経費該当性の判断

```
以下の支出が個人事業主（Webデザイン業・青色申告）の事業経費として
認められるか判断してください。
認められる場合は勘定科目と按分率も提示してください。

1. 自宅兼事務所の家賃 月12万円（作業スペース30%）
2. カフェでの作業時のコーヒー代 480円
3. デザイン系の書籍 3,200円
4. クライアントとの忘年会 1人あたり6,500円（参加者5名）
5. 事業用スマホの通信費 月8,800円（私用20%）
```

### ChatGPTの回答例

> 1. **家賃**: 経費可。「地代家賃」で按分30% = 36,000円/月
> 2. **カフェ代**: 経費可。「会議費」として計上。ただし頻度が高すぎると否認リスクあり
> 3. **書籍**: 経費可。「新聞図書費」3,200円。事業関連性が明確
> 4. **忘年会**: 1人5,000円超のため「交際費」。個人事業主は全額経費可（法人の損金不算入規定は適用外）
> 5. **通信費**: 経費可。「通信費」で按分80% = 7,040円/月

**CPA試験合格者監修コメント**:
ChatGPTの回答は概ね正確ですが、2点補足します。まず、カフェ代は「1人での作業」であれば会議費ではなく雑費が適切です。会議費は複数人での打ち合わせが前提です。また、忘年会の交際費は「事業関連の取引先との飲食」であることを証明する記録（参加者名・目的）が必要です。ChatGPTの回答を鵜呑みにせず、このような細かい要件を自分でも確認してください。

## 活用法③：確定申告書の最終チェック

freeeで作成した確定申告書（青色申告決算書・確定申告書B）の記載内容をChatGPTにチェックしてもらう方法です。

### プロンプト例：申告書の整合性チェック

```
以下は2025年分の青色申告決算書の損益計算書データです。
数値の整合性と、見落としがちな控除・特例がないかチェックしてください。

【損益計算書】
売上: 8,500,000円
売上原価: 0円（サービス業）
経費合計: 3,200,000円
  - 通信費: 105,600円
  - 地代家賃: 432,000円
  - 減価償却費: 298,000円
  - 外注費: 1,500,000円
  - その他: 864,400円
青色申告特別控除前所得: 5,300,000円
青色申告特別控除: 650,000円

【確認事項】
- e-Taxで電子申告予定
- 小規模企業共済に月額7万円加入
- 国民健康保険料 年額52万円
- ふるさと納税 年額8万円
```

### ChatGPTの回答例

> **整合性チェック結果**:
> - 売上 - 経費 = 5,300,000円 → OK
> - 青色申告特別控除65万円 → e-Tax + 複式簿記が前提、条件を満たしていればOK
>
> **見落とし指摘**:
> 1. 小規模企業共済 月7万円 × 12ヶ月 = 84万円 → 「小規模企業共済等掛金控除」で所得控除に計上されていますか？
> 2. 国民健康保険料52万円 → 「社会保険料控除」に計上されていますか？
> 3. ふるさと納税8万円 → 「寄附金控除」で自己負担2,000円を除く78,000円が控除対象
> 4. 基礎控除48万円は計上されていますか？

**CPA試験合格者監修コメント**:
ChatGPTによる「見落としチェック」は非常に有効です。実際に申告書作成時に忘れがちなのが小規模企業共済や生命保険料控除の計上漏れです。freeeの確定申告機能は控除項目を入力すれば自動計算しますが、そもそも入力を忘れていると反映されません。申告前にChatGPTで「見落とし控除がないか」を確認する習慣をつけましょう。

## ChatGPT活用時の注意点

| 注意点 | 詳細 |
|--------|------|
| 機密情報の取扱い | 個人名・マイナンバー・口座番号はChatGPTに入力しない |
| 回答の精度限界 | 税法改正や最新通達を反映していない可能性がある |
| 一次確認用途 | 金額が大きい取引や特殊な税務処理は必ず税理士に確認 |
| バージョン差異 | ChatGPT無料版よりPlusの方が税務回答の精度が高い傾向 |
