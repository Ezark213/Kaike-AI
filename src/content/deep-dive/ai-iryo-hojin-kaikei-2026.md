## 背景：医療法人会計の特殊性

医療法人の会計は、一般企業の会計基準に加えて「医療法人会計基準」「病院会計準則」に従う必要があります。

**医療法人特有の論点**:

| 論点 | 一般企業との違い |
|------|----------------|
| 損益計算書 | 四段階区分（本来業務/附帯業務/収益業務/特別損益） |
| 勘定科目 | 医業収益/医業費用の独自科目体系 |
| 事業報告書 | 都道府県への年次報告義務 |
| 剰余金の配当 | 原則禁止（出資持分あり法人は例外あり） |
| MS法人との取引 | 関連当事者取引の適正性が論点 |
| 消費税 | 社会保険診療は非課税売上 |

## AI活用の実践手法

### 1. 四段階区分損益計算書の作成

```
以下の医療法人の収支データを四段階区分の損益計算書に組み替えてください。

【医療法人概要】
- 法人名: 医療法人社団○○会
- 施設: 診療所1施設（内科・整形外科）
- 付帯事業: 訪問看護ステーション

【収支データ（年間）】
■ 収入
- 保険診療収入: ¥180,000,000
- 自費診療収入: ¥20,000,000
- 訪問看護収入: ¥15,000,000
- 駐車場賃貸収入: ¥2,400,000
- 受取利息: ¥50,000
- 補助金収入: ¥5,000,000

■ 支出
- 医師・看護師人件費: ¥90,000,000
- 医薬品費: ¥25,000,000
- 医療材料費: ¥15,000,000
- 給食費: ¥3,000,000
- 減価償却費（医療機器）: ¥10,000,000
- 訪問看護人件費: ¥8,000,000
- 訪問看護経費: ¥2,000,000
- 管理部門人件費: ¥12,000,000
- 施設維持費: ¥8,000,000
- その他管理費: ¥5,000,000

【作成してほしい成果物】
1. 四段階区分の損益計算書
2. 各収支項目の区分判定根拠
3. 本来業務損益・附帯業務損益・全体損益の算出
```

### 2. 社会保険診療報酬の概算経費率

```
以下の個人医療機関のデータで、概算経費率の適用可否と計算を行ってください。

【データ】
- 形態: 個人開業医（内科）
- 社会保険診療報酬: ¥45,000,000
- 自費診療収入: ¥8,000,000
- 実際の必要経費: ¥28,000,000

【確認してほしい項目】
1. 概算経費率の適用要件（社保収入5,000万円以下）の充足確認
2. 概算経費率による必要経費の計算
3. 実額経費との有利不利判定
4. 確定申告書への記載方法
5. 消費税の課税売上割合への影響
```

### 3. 事業報告書の作成支援

```
以下のデータから、都道府県に提出する医療法人の事業報告書の概要部分を作成してください。

【基本情報】
- 法人名: 医療法人社団○○会
- 開設する施設: ○○クリニック（無床診療所）
- 理事長: △△ △△
- 決算期: 3月31日
- 従業員数: 常勤15名、非常勤8名

【事業実績】
- 外来患者数: 年間28,000人（1日平均115人）
- 手術件数: 年間120件
- 主な診療科: 内科、整形外科、リハビリテーション科

【財務サマリー】
- 事業収益: ¥220,000,000
- 事業費用: ¥195,000,000
- 当期純利益: ¥25,000,000
- 総資産: ¥150,000,000

【作成してほしい項目】
1. 事業の概況
2. 施設の状況
3. 業務の状況（患者数推移・診療科別実績）
4. 会計の状況（要約）
5. 役員の状況
```

### 4. MS法人との取引の適正性チェック

```
以下の医療法人とMS法人（メディカルサービス法人）との取引について、
適正性をチェックしてください。

【関係】
- 医療法人社団○○会（理事長: A氏）
- 株式会社○○メディカルサービス（代表取締役: A氏の配偶者）

【MS法人との取引】
1. 医療事務の委託: 月額¥800,000（スタッフ5名派遣）
2. 医療機器のリース: 月額¥300,000（CT装置）
3. 清掃業務委託: 月額¥150,000
4. 薬品の仕入代行: 実費+手数料5%

【チェックしてほしい項目】
1. 各取引の適正価格との比較（市場価格の目安）
2. 医療法第54条（剰余金配当禁止）との関係
3. 法人税法上の問題点（寄附金認定リスク）
4. 開示すべき関連当事者取引の範囲
5. 改善提案
```
