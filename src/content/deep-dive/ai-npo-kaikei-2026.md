## 背景：NPO・社団法人の会計の課題

NPO法人や一般社団法人の会計は、営利企業とは異なる独自の会計基準に従う必要があります。しかし、多くの非営利法人では専任の経理担当者がおらず、理事や事務局長が「手探りで」会計処理を行っているのが現状です。

**非営利法人の会計で特に困る点**:

| 課題 | 詳細 |
|------|------|
| 活動計算書の作成 | 事業費・管理費の区分が必要 |
| 事業費と管理費の按分 | 共通経費の按分基準の設定 |
| 助成金の会計処理 | 受領時・使用時・返還時の処理 |
| 使途報告書の作成 | 助成金ごとの支出明細の集計 |
| 所轄庁への事業報告書 | 年度ごとの提出書類の準備 |
| ボランティアの労務費評価 | 無償労働の金銭換算（任意） |

## AI活用の実践手法

### 1. 事業費・管理費の按分計算

```
以下のNPO法人の経費データから、事業費と管理費への按分計算を行ってください。

【NPO法人概要】
- 事業: 子育て支援事業（事業A）、高齢者サポート事業（事業B）
- スタッフ: 常勤3名、パート2名

【共通経費（按分対象）】
| 経費項目 | 年間金額 | 按分基準 |
|---------|---------|---------|
| 事務所賃料 | ¥1,200,000 | 占有面積比 |
| 通信費 | ¥180,000 | 従事時間比 |
| 消耗品費 | ¥300,000 | 従事時間比 |
| 理事長人件費 | ¥3,600,000 | 従事時間比 |
| 水道光熱費 | ¥240,000 | 占有面積比 |

【按分基準データ】
| 基準 | 事業A | 事業B | 管理 |
|------|-------|-------|------|
| 占有面積 | 40㎡ | 30㎡ | 10㎡ |
| 従事時間 | 40% | 35% | 25% |

【作成してほしい成果物】
1. 按分計算表（経費×按分基準→事業A/事業B/管理の金額）
2. 活動計算書の経常費用の部（事業費・管理費の内訳）
3. 按分基準の根拠説明（所轄庁への説明用）
```

### 2. 助成金の会計処理

```
以下の助成金の会計処理を教えてください。

【助成金情報】
- 助成元: ○○財団
- 助成金額: ¥2,000,000
- 助成期間: 2026年4月〜2027年3月
- 使途: 子育て支援事業の運営費
- 前払い: 2026年4月に全額入金済み
- 報告義務: 期末に使途報告書を提出
- 残金の取り扱い: 未使用分は返還

【実績（2026年4月〜12月）】
- 使用額: ¥1,500,000
- 残額: ¥500,000（うち¥200,000は1〜3月に使用予定、¥300,000は返還見込み）

【回答に含めてほしい項目】
1. 受領時の仕訳（NPO会計基準に準拠）
2. 期中の使用時の仕訳
3. 期末の処理（返還見込み分の処理）
4. 活動計算書での表示方法
5. 使途報告書のフォーマット案
```

### 3. 所轄庁提出書類の作成支援

```
以下のNPO法人の年度データから、所轄庁に提出する事業報告書のドラフトを作成してください。

【法人情報】
- 名称: 特定非営利活動法人○○サポート
- 設立: 2020年4月
- 所轄庁: △△県
- 事業年度: 4月〜3月

【年度実績（2025年度）】
- 事業収益: ¥8,000,000（うち助成金¥3,000,000）
- 事業費用: ¥7,500,000
- 管理費用: ¥800,000
- 当期正味財産増減額: -¥300,000
- 受益者数: 年間350名
- イベント開催: 24回
- ボランティア参加者: 延べ150名

【事業報告書に必要な項目】
1. 事業の概要
2. 事業の実施状況（数値目標と実績）
3. 収支の概要
4. 理事の異動
5. 社員の状況
```
