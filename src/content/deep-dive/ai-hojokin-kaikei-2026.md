## 背景：補助金会計処理の複雑さ

中小企業が活用できる補助金・助成金は年間数千種類ありますが、会計処理を正確に行えている企業は多くありません。

**よくある失敗**:

| 失敗パターン | 影響 |
|------------|------|
| 入金時に全額を収益計上 | 圧縮記帳の機会を逃し税負担増 |
| 圧縮記帳の適用要件を満たさない | 税務否認→追徴課税 |
| 実績報告書の経費区分ミス | 補助金の返還請求 |
| 消費税の仕入税額控除の二重取り | 補助金返還+加算金 |

## 主要補助金の会計処理

### 1. 圧縮記帳の仕訳

```
以下の補助金取引について、圧縮記帳の仕訳を作成してください。

【取引内容】
- 補助金名: ものづくり補助金
- 対象設備: 製造用NC旋盤
- 設備取得価額: ¥10,000,000（税抜）
- 補助金交付額: ¥5,000,000（補助率1/2）
- 設備取得日: 2026年7月1日
- 補助金入金日: 2026年10月15日
- 耐用年数: 10年（定額法）
- 決算期: 3月31日

【作成してほしい仕訳】
1. 設備取得時の仕訳
2. 補助金入金時の仕訳
3. 圧縮記帳の仕訳（直接減額方式）
4. 決算時の減価償却仕訳（圧縮後の取得価額ベース）
5. 圧縮記帳を適用しない場合との税額比較
6. 積立金方式の場合の仕訳（比較）
```

### 2. 収益計上タイミングの判定

```
以下の3つの補助金について、収益計上のタイミングを判定してください。

【ケース1: 事前交付型】
- IT導入補助金（事前に交付決定→設備導入→実績報告→確定→入金）
- 交付決定日: 2026年4月
- 設備導入: 2026年6月
- 実績報告: 2026年8月
- 確定通知: 2026年9月
- 入金: 2026年10月
- 決算期: 3月末

【ケース2: 概算交付→精算型】
- 雇用調整助成金（概算で交付→実績に基づき精算）
- 概算交付: 月額¥500,000（6ヶ月間）
- 精算結果: ¥2,800,000（差額¥200,000返還）

【ケース3: 後払い型】
- 事業再構築補助金（事業完了後に入金）
- 事業実施期間: 2026年4月〜2027年3月
- 入金見込み: 2027年6月
- 決算期をまたぐケース

【各ケースについて回答】
1. 収益計上すべき時点（発生主義 vs 実現主義）
2. 決算期末の仕訳（未収入金の計上要否）
3. 消費税の取り扱い
4. 注意すべきポイント
```

### 3. 実績報告書の作成支援

```
以下のデータから、ものづくり補助金の実績報告書の経費明細を作成してください。

【交付決定内容】
- 補助事業名: AI検査システム導入による品質管理強化
- 補助金額: ¥5,000,000
- 補助対象経費: 機械装置費、技術導入費、クラウドサービス利用費

【実績データ】
| No | 日付 | 内容 | 金額（税抜） | 区分 | 証憑 |
|----|------|------|-----------|------|------|
| 1 | 6/1 | AI検査カメラ購入 | ¥3,000,000 | 機械装置費 | 請求書#A001 |
| 2 | 6/15 | AI検査ソフトウェアライセンス | ¥1,500,000 | 技術導入費 | 請求書#B001 |
| 3 | 7/1 | AWS利用料（6ヶ月分） | ¥300,000 | クラウド利用費 | 請求書#C001 |
| 4 | 7/15 | 設置工事費 | ¥500,000 | 機械装置費 | 請求書#A002 |
| 5 | 8/1 | 社員研修費（AI操作） | ¥200,000 | — | 請求書#D001 |

【確認してほしい項目】
1. 各経費の補助対象可否（研修費は対象外の可能性）
2. 経費区分の正確性
3. 補助対象経費の合計と補助金額の整合性
4. 消費税の取り扱い（仕入税額控除との関係）
5. 実績報告書の記載例
```

### 4. 消費税の取り扱い

```
補助金受給時の消費税の取り扱いについて整理してください。

【前提】
- 当社: 消費税課税事業者（本則課税）
- 補助金: IT導入補助金 ¥2,000,000
- 補助対象経費: ソフトウェア ¥4,000,000（税込¥4,400,000）

【確認したい項目】
1. 補助金収入は消費税の課税対象か（不課税?）
2. 補助対象経費の仕入税額控除は可能か
3. 「補助金に係る消費税仕入控除税額の報告」の要否
4. 簡易課税制度適用事業者の場合の取り扱い
5. 実務上の注意点（二重取りにならないために）
```
