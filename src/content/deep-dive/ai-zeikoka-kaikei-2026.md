## 要点まとめ（30秒で読める）

- 税効果会計はAI活用の効果が最も高い分野の一つ — **一時差異の集計・税率適用の計算を90%自動化**可能
- ChatGPTで**一時差異の分類（将来減算/加算）を自動判定**し、仕訳を自動生成できる
- 最も判断が難しい「回収可能性の評価」もAIが**企業分類（1〜5）の判定材料を整理**してくれる
- **注意**: 回収可能性の最終判断は会計士・税理士の専門的判断が不可欠

---

## 背景：税効果会計はなぜ難しいのか

税効果会計は、会計上の利益と税務上の所得の差異を調整する会計処理です。多くの経理担当者が「最も苦手な決算論点」として挙げる分野です。

**難しさの3つの原因**:
1. **概念の抽象性**: 「将来の税金の前払い/未払い」という考え方が直感的でない
2. **税法との二重基準**: 会計基準と税法の両方の知識が必要
3. **将来予測の必要性**: 回収可能性の判断に将来の課税所得の見積りが必要

## AI税効果会計の実践プロンプト集

### 1. 一時差異の分類と集計

```
以下の決算データから、税効果会計に必要な一時差異を分類・集計してください。

【決算データ】
1. 貸倒引当金: 会計上¥5,000,000計上、税務上損金算入限度額¥2,000,000
2. 賞与引当金: 会計上¥10,000,000計上、税務上は支払時に損金算入
3. 減価償却費: 会計上¥8,000,000、税務上¥6,000,000（会計が超過）
4. 有価証券評価損: 会計上¥3,000,000計上、税務上は売却時まで損金不算入
5. 未払事業税: 会計上¥4,000,000計上、税務上は申告時に損金算入
6. 受取配当金の益金不算入: ¥1,000,000
7. 交際費の損金不算入: ¥2,000,000（永久差異）

【分類してほしい項目】
1. 各項目を「将来減算一時差異」「将来加算一時差異」「永久差異」に分類
2. 一時差異の残高一覧表を作成
3. 法定実効税率30%での繰延税金資産・繰延税金負債の金額
4. 相殺後の純額
5. 仕訳（前期末残高ゼロと仮定）
```

### 2. 回収可能性の評価

```
以下のデータに基づき、繰延税金資産の回収可能性を評価してください。

【企業概要】
- 業種: 製造業
- 売上高: ¥500,000,000
- 資本金: ¥50,000,000

【過去5期の課税所得】
| 期 | 課税所得 | 特殊要因 |
|----|---------|---------|
| X1期 | ¥30,000,000 | — |
| X2期 | ¥25,000,000 | — |
| X3期 | -¥5,000,000 | 工場移転費用 |
| X4期 | ¥20,000,000 | — |
| X5期(当期) | ¥28,000,000 | — |

【将来の課税所得見込み】
- X6期〜X10期: 年¥25,000,000〜30,000,000（安定見込み）

【繰延税金資産の内訳】
- 将来減算一時差異合計: ¥22,000,000
- 繰延税金資産（税率30%）: ¥6,600,000
- 税務上の繰越欠損金: ¥5,000,000（X3期発生、繰越期限X13期）

【評価してほしい項目】
1. 企業分類（1〜5のどれに該当するか）
2. 回収可能性の判定結果
3. 計上すべき繰延税金資産の金額
4. スケジューリングの要否
5. 監査人への説明ポイント
```

### 3. 税率変更時の処理

```
以下の条件で、税率変更に伴う税効果会計の処理を計算してください。

【条件】
- 現行の法定実効税率: 30%
- 改正後の法定実効税率: 28%（X7期から適用）
- 繰延税金資産残高（変更前）: ¥6,600,000（一時差異¥22,000,000×30%）

【一時差異の解消時期】
| 一時差異 | 金額 | 解消時期 |
|---------|------|---------|
| 貸倒引当金 | ¥3,000,000 | X6期（旧税率） |
| 賞与引当金 | ¥10,000,000 | X6期（旧税率） |
| 減価償却超過額 | ¥2,000,000 | X7期（新税率） |
| 有価証券評価損 | ¥3,000,000 | X8期（新税率） |
| 未払事業税 | ¥4,000,000 | X6期（旧税率） |

【計算してほしい項目】
1. 新税率適用後の繰延税金資産の金額
2. 税率変更による影響額
3. 仕訳（法人税等調整額の計上）
4. 開示上の注記事項
```

### 4. 連結上の税効果

```
以下のグループ間取引に係る税効果会計の処理を計算してください。

【条件】
- 親会社A社が子会社B社に商品を販売
- B社の期末在庫に含まれる未実現利益: ¥4,000,000
- A社の法定実効税率: 30%
- B社の法定実効税率: 30%

【計算してほしい項目】
1. 連結上の未実現利益消去仕訳
2. 税効果の仕訳（繰延税金資産の計上）
3. 翌期の戻入仕訳
4. 税率が異なる場合（A社30%、B社25%）の処理
```

## 税効果会計のチートシート

### 主要な一時差異の分類

| 項目 | 分類 | 繰延税金 |
|------|------|---------|
| 貸倒引当金（税務超過分） | 将来減算 | 資産 |
| 賞与引当金 | 将来減算 | 資産 |
| 退職給付引当金 | 将来減算 | 資産 |
| 減価償却超過額 | 将来減算 | 資産 |
| 有価証券評価損 | 将来減算 | 資産 |
| 未払事業税 | 将来減算 | 資産 |
| 繰越欠損金 | 将来減算 | 資産 |
| 資産の評価益（税務上計上済） | 将来加算 | 負債 |
| 圧縮積立金 | 将来加算 | 負債 |

### 永久差異（税効果なし）

- 交際費の損金不算入額
- 受取配当金の益金不算入額
- 寄附金の損金不算入額
- 住民税均等割
- 罰科金
