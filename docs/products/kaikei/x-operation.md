- 投稿間隔: 最低2時間空ける

- アフィリエイトリンクは1日1回まで

## 12. @Ezark_ct 連携ポリシー（2026-02-21追加）

### 方針: Kaikei AI Daily更新を @Ezark_ct（会社公式）でも周知

**Kaikei AI Dailyがデプロイされるたびに、イザークコンサルティング公式アカウント @Ezark_ct でも告知ツイートを投稿する。**

```
連携フロー:
site-publisher.sh（デプロイ）
    ↓
@Ezark_ct ツイートドラフト自動生成
(content/kaikei-ai/x-posts/ezark-ct-YYYY-MM-DD.md)
    ↓
OpenClaw キューに追加 (tasks/openclaw-queue.md)
    ↓
OpenClaw が @Ezark_ct から投稿
```

### @Ezark_ct 投稿ルール（Kaikei AI Daily告知用）

| 項目 | 設定 |
|------|------|
| 投稿頻度 | 毎日（Kaikei AI Daily更新と連動） |
| 投稿時間 | デプロイ後できるだけ早く（目標: 朝6時〜8時台） |
| 文字数 | 280文字以内 |
| ハッシュタグ | `#AI会計 #経理DX #会計ニュース` |
| URL | `https://kaikei-ai.jp/daily/YYYY-MM-DD` |
| PR表記 | 不要（自社メディアの更新告知） |

### テンプレート

```
【会計AI最新情報】{YYYY年MM月DD日}

Kaikei AI Dailyを更新しました。

本日のトピック:
- {記事タイトル1}
- {記事タイトル2}

CPA試験合格者監修・毎朝更新
https://kaikei-ai.jp/daily/{DATE}

#AI会計 #経理DX #会計ニュース
```

## 13. 法的注意事項

1. **アフィリエイト表記**: 広告リンクには「PR」「広告」と明記
2. **税務助言の免責**: 具体的な税務判断を促す投稿は避ける
3. **CPA名称**: 「CPA試験合格者監修」と正確に表記
4. **X利用規約**: 自動化ルール（API利用規約）を遵守
5. **著作権**: ニュースの引用は適正範囲で
