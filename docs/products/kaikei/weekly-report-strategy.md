## 概要

毎週日曜日に、AI×会計の深掘り記事をnote.comで公開。
無料部分（70%）で読者を集め、有料部分（30%）でプロンプト・実装ガイド等の実践的価値を提供。

## 収益モデル

| チャネル | 単価 | 想定数 | 月間収益 |
|---------|------|--------|---------|
| 有料記事（単品） | ¥500/本 | 20-50購入/月 | ¥10,000-25,000 |
| メンバーシップ | ¥980/月 | 10-30人 | ¥9,800-29,400 |
| BOOTH誘導（間接） | - | - | 売上増への寄与 |
| **合計** | | | **¥20,000-55,000** |

## USP（差別化ポイント）

1. **CPA試験合格者の実務視点** -- 他のAI系noteにない会計専門家の解説
2. **週次の定期刊行物** -- 毎週読めるリズム感、雑誌的な体験
3. **無料部分の充実** -- 70%無料で「有料部分も読みたい」を生む構成
4. **実装可能な有料部分** -- プロンプト・手順書等、すぐ使える実践的コンテンツ

## 自動化パイプライン

### フロー

```
[Debian Cron 毎週日曜 16:00 JST]
    │
    ▼
scripts/kaikei/weekly-report-generator.py
    ├── 今週のKaikei AI Daily記事を集約
    │   └── content/kaikei-ai/daily/ から当週分を読み込み
    ├── WebSearchで補完情報取得
    ├── Anthropic APIで3,000-4,000字記事ドラフト生成
    │   ├── 無料部分（ニュースTOP5 + 深掘り解説）
    │   └── 有料部分（実装ガイド + リスク分析 + 来週予告）
    └── content/kaikei-ai/weekly-reports/YYYY-WXX.md に保存
    │
    ▼
[Google Chat通知]
    └── CPA合格者に「レビュー依頼」通知
    │
    ▼
[CPA合格者レビュー（手動）]
    ├── ファクトチェック
    ├── 実務視点の加筆
    ├── 有料部分の充実化
    └── 公開OKの判断
    │
    ▼
[OpenClaw]
    ├── note.comに下書き投稿
    ├── サムネイル設定
    ├── 有料設定（¥500）
    └── CPA合格者の最終確認後に公開
    │
    ▼
[プロモーション]
    ├── @KaikeiAI_Bot で告知ツイート
    └── Kaikei AI Dailyサイトにリンク追加
```

| リスク | 影響度 | 対策 |
|--------|--------|------|

| CPA合格者のレビュー遅延 | 中 | 日曜16時生成→月曜夜までにレビュー→火曜公開 |
| ネタ切れ | 中 | 年間テーマ計画を先行作成、読者リクエスト募集 |
| note有料記事の競合増加 | 中 | CPA試験合格者監修のUSPで差別化 |
| API コスト増 | 低 | 週1回の生成なので月¥1,000-2,000程度 |

## 法的注意事項

1. **AI生成表記**: 「AIによる自動収集・要約をベースに、CPA試験合格者が監修・加筆」と明記
2. **免責事項**: 全記事末尾に「具体的な会計・税務判断は公認会計士・税理士にご相談ください」
3. **有料記事の返金ポリシー**: noteのデフォルト（購入後返金不可）に従う
4. **CPA名称**: 「CPA試験合格者」と正確に表記
5. **プロンプトの効果保証**: 「効果を保証するものではない」と明記
