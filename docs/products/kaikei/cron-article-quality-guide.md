# Kaikei AI Daily -- Cron生成記事 品質ガイド

**作成日**: 2026-03-21
**目的**: cronで自動生成される日刊記事に必須の品質要素を定義し、手動レビューなしで収益最大化できる記事品質を担保する

---

## 必須品質要素（現在欠けているもの）

### 1. インラインCTA（記事本文中のアフィリエイトリンク）

記事本文の中で、関連する会計SaaSへのアフィリエイトリンクを自然に挿入する。末尾のCTAブロックだけでなく、本文中に溶け込ませることでCTR向上。

**配置ルール**:
- 記事中盤（2-3本目のニュースの後）に1箇所
- 記事末尾のCTAブロックに1箇所
- freee関連ニュースにはfreeeリンク、MF関連にはMFリンクを対応させる

**テンプレート例**:

```markdown
> この機能を試してみたい方は、[freee会計の無料プラン](アフィリエイトURL)で実際にAI仕訳を体験できます。

> クラウド会計の導入を検討中の方は、[マネーフォワード クラウド](アフィリエイトURL)の無料トライアルがおすすめです。
```

### 2. Value AI Writer リンク

AI記事作成ツールとしてValue AI Writerのアフィリエイトリンクを記事末尾に配置する。

**固定リンク**:
```
https://px.a8.net/svt/ejp?a8mat=4AZ9C5+6Y24EA+1JUK+1HL85U
```

**テンプレート例**:

```markdown
---

**PR**: AI×会計の最新情報を効率的に発信したい方へ

会計ブログの記事作成を効率化するなら [Value AI Writer](https://px.a8.net/svt/ejp?a8mat=4AZ9C5+6Y24EA+1JUK+1HL85U) がおすすめです。SEO対策済みの記事をAIが自動生成します。
```

### 3. PR表記（「広告」ではなく「PR」）

アフィリエイトリンクを含む記事には、記事冒頭に「PR」表記を配置する。景品表示法・ステマ規制対応。

**配置ルール**:
- 記事タイトル直下に `*PR: 本記事にはアフィリエイトリンクが含まれます*` を配置
- 「広告」ではなく「PR」を使用（業界慣行に合わせる）

**テンプレート例**:

```markdown
# 【AI×会計ニュース】2026年3月21日

*PR: 本記事にはアフィリエイトリンクが含まれます*

---

## 本日のAI会計ニュース
```

### 4. 5本構成（3本ではなく）

1日のニュース記事は**5本のニュースピックアップ**で構成する。3本では情報量不足で読者の再訪率が低下する。

**構成テンプレート**:

```markdown
## 本日のAI会計ニュース 5選

### 1. [見出し] -- 最重要ニュース
（要約 + 実務影響）

### 2. [見出し] -- 重要ニュース
（要約 + 実務影響）

### 3. [見出し] -- 注目ニュース
（要約 + 実務影響）

--- インラインCTA挿入ポイント ---

### 4. [見出し] -- トレンドニュース
（要約 + 実務影響）

### 5. [見出し] -- 海外・技術動向
（要約 + 実務影響）
```

**`feeds.yaml` の `selection_criteria` との整合**:
- `max_articles_per_day: 5` に設定済み（現行一致）
- `min_articles_per_day: 3` は最低保証値として維持（ニュースが少ない日のフォールバック）

### 5. BOOTH CTA具体化（商品名明記）

BOOTH商品へのCTAは「関連商品はこちら」のような曖昧な表現ではなく、商品名と内容を明記する。

**テンプレート例**:

```markdown
---

**関連BOOTH商品**

本日のニュースをより深く理解するために:

- [AI仕訳マスター：freee×ChatGPT実践ガイド](BOOTHリンク) -- freeeでのAI仕訳設定を完全解説
- [経理AI導入チェックリスト 2026年版](BOOTHリンク) -- 導入判断に必要な50項目を網羅
```

**ルール**:
- 商品名をそのまま記載（「関連商品」NG）
- 商品の具体的なメリットを1行で説明
- 記事内容と関連性の高い商品のみ掲載（最大2-3点）

### 6. セルフリプライURL付きのX投稿ドラフト

記事公開時のX（Twitter）投稿は、メインツイート + セルフリプライの2段構成とする。メインツイートにURLを含めない（CLAUDE.mdルール25準拠）。

## チェックリスト（cron生成後の自動検証用）

記事生成後、以下を自動検証すること:

- [ ] PR表記がタイトル直下にあるか
- [ ] ニュースが5本（最低3本）含まれているか
- [ ] 各ニュースに「実務への影響」セクションがあるか
- [ ] インラインCTAが記事中盤に1箇所あるか
- [ ] BOOTH CTAに具体的な商品名が入っているか
- [ ] Value AI Writerリンク（a8mat=4AZ9C5+6Y24EA+1JUK+1HL85U）が末尾にあるか
- [ ] X投稿ドラフト（メイン + セルフリプライ）がコメントとして含まれているか
- [ ] メインツイートにURLが含まれていないか（ルール25準拠）
- [ ] 免責事項が末尾にあるか
